# LICENSES & Attribution

This file centralizes the authoritative license texts and a short mapping
for the repository. Per-directory `LICENSE` or `BRANDING_LICENSE` files
should be short notices that point to this file.

# LICENSES & Mapping

Authoritative license texts for this repository live at the repository
root:

- MPL-2.0 (code): `../LICENSE`
- Apache-2.0 (protocol/docs): `../APACHE-2.0`

Per-directory `LICENSE` or `BRANDING_LICENSE` files should be short
notices that reference these root files and include a single `SPDX-License-Identifier`.

Example `frontend/LICENSE` content:

    Mozilla Public License 2.0
    See: ../LICENSE
    SPDX-License-Identifier: MPL-2.0

Example `docs/` protocol header:

    SPDX-License-Identifier: Apache-2.0

If you need the complete canonical text, consult the files at the repo
root (`LICENSE` and `APACHE-2.0`) or the official online sources.
